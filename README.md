能够设置v4优先的 Shadowsocks 和 SOCKS5 安装管理脚本，基于Xray
```
bash <(curl -Ls https://raw.githubusercontent.com/cyeinfpro/Shadowsocks-v4first/refs/heads/main/Shadowsocks-v4first.sh)
```
请选择操作：
1. 安装 Shadowsocks
2. 修改 Shadowsocks 配置
3. 安装 SOCKS5
4. 修改 SOCKS5 配置
5. 卸载已安装代理
6. 修改 IPv4 优先设置
7. 进行网络优化
8. 显示代理的配置信息
0. 退出
